To work on this project, we used data coming from public datasets.
As most of them requested, we cite our sources and the people that worked on them, giving us the chance of visualizing clean and truthful information.

Methodology

This website was created via multiple tools:

  • HTML, CSS, JavaScript, Material UI baseline for the front-end
  • D3.js for the creation of the plots
  • Python notebook, together with Numpy, Pandas and Matplotlib libraries, for the preprocessing of the data


The majority of the data that we used was usable as a csv file, after some sort of preprocessing.
